Log 4: CSS Color

What do these CSS properties do?

How does CSS accept color as a value? Provide examples of colors in these styles:
  • Named

    • These are simple color names that CSS already recognizes, such as red, blue, green, or coral. Example: color: red;

  • Hexadecimal

    • These use a “#” followed by numbers and letters that represent the red, green, and blue values of a color. Example: background-color: #00ff00; (green)

  • RGB

    • This format mixes red, green, and blue values, each between 0 and 255. Example: border-color: rgb(255, 0, 0); (red)

  • HSL

    • This stands for hue, saturation, and lightness, which define color based on its position on the color wheel. Example: color: hsl(120, 100%, 50%); (green)
